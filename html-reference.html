HTML <html>

Description:

The html tag is used in all html files. The tag defines the document as a html document.

Syntax:

The html tag is like this: <html>. The start tag is <html> and the end the end tag is </html>. The start tag is placed at the beginning of the document right after the <!DOCTYPE> command. The end tag is located after every single code at the bottom of the file.

Example:

<DOCTYPE>
<html>
(body)
</html>

Display:

The html element everything coded onto the page. Images, Links, text, and etc are all html incorporated on the page.

Head <head>

Description:

The head is a container for metadata. The metadata defines the components of the page.

Syntax:

In html text, the head tag is shown after the html tag and is right before the body tag. The start tag is <head> and the end tag is </head>.

Example:

<!DOCTYPE>
<html>
<head>
(metadata)
</head>
</html>

Display:

The head element helps define anything in the body. Therefore, the head can help display styles such as certain lines, colors, fonts, and even sizes. Basically, it decorates the page with many creative structures.

Title <title>

Description:

The title element is used to name the website page.

Syntax:

The title element is located within the head element. The start tag is <title>.

Example:

<!DOCTYPE>
<html>
<head>
<title>
(name)
</title>
</head>
</html>

Display:

The title displays the name at the tab area of your own browser.

Body <body>

Description:

The body element is where you put all the text, images, and attributes. It is the main structure of all the attributes.

Syntax:

The body element is located in the html element and after the head element.

Example:

<!DOCTYPE>
<html>
<body>
(attributes)
</body>
</html>

Display:

The body shows all the text, information, and content.

Headings <h1>, <h2>, <h3>

Description:

Headings are mini-titles or section indicators within a website. They help indicate the name or topic of each part. There are different types of headers there are size 1,2,3, and etc.

Syntax:

Headings are used inside the body element and usually are utilized right before a paragraph element, image, or table.

Example:

<!DOCTYPE>
<html>
<body>
<h1>
(title)
</h1>
</body>
</html>

Display:

The headers show a bolded, enlarged word like this sentence.

Paragraph <p>

Description:

Paragraphs are small text in which you can type descriptions, explanations, and etc.

Syntax:

Paragraphs are within the body element and are usually follows a heading.

Example:

<!DOCTYPE>
<html>
<body>
<h1>
</h1>
<p>
(text)
</p>
</body>
</html>

Display:

Paragraphs are very small, unbolded text. They can fit inside websites as long text.

Breaks <br>

Description:

Breaks are used to make a new line for text.

Syntax:

Breaks can be used within just about any attribute with text inside. This occurs in the body element.

Example:

<!DOCTYPE>
<html>
<body>
<p>(text)<br>(extending text)</p>
</body>
</html>

Display:

Breaks make the text go to a new line, so it will indent
the text after the element below the text before the element.

The lines element is usually used to separate two different topics or sections of the page.

Example:

<html>
<body>
<h1>
</h1>
<hr>
<h2>
</h2>

Display:

For separation, a line just appears between two groups of attributes. Just like the separating line below.

Comments <!--(text)-->

Description:

Comments is used to exclude code or unnecessary text.

Syntax:

Comments could be used anywhere to make it not part of a code. Usually it is used as instructions or a note before a code.It starts as <!--(text)--> in the html file and /*(text)*/ in the css file.

Example:

<!DOCTYPE>
<html>
<body>
<!---
<h1>
(text)
</h1>
--->
</body>
</html>

Display:

Comments do not appear. The only way you can see the comments is in the code itself. They comments are green text inside the code.

Description:

Links are used as a click-on to get to other websites.

Syntax:

Links could be used anywhere in the website. They are mostly used when the coder wants the visitor to go somewhere.

Example:

<!DOCTYPE>
<html>
<body>
<h1>
</h1>
<a href = "url">link text</a>
</body>
</html>

Display:

Text will appear in a blue color which stands for a link.

Images <img>

Description:

Images are pictures put onto the website. They are colorful and very good descriptors.

Syntax:

Images are sometimes used to describe things or provide a vivid picture. Other times they can be used as links. The images element can be placed anywhere in the body element.

Example:

<!DOCTYPE>
<html>
<body>
<img src="(img.jpg)" alt=(image name)>
</body>
</html>

Display:

Pictures are very clear and amazing when displayed on websites. They form more decorations and art.

Lists <ul>, <ol>, <li>

Description:

Lists are used to arrange items in an orderly fashion. Lists can be numbered, dotted, or alphabetical.

Syntax

Lists are used within the body. Lists can be unnumbered or and ordered. For non-numbered, lists must be in the unnumbered tag(ul). For numbered, they must be in the ordered tag(ol). The li tag is the item, and you can make many of these inside lists. There are many other list tags on w3schools.

Example:

<!DOCTYPE>
<html>
<body>
<ul>
<li>
(item)
</li>
</ul>
</body>
</html>

Display:

Lists arrange items just like any list. It lists items from top to bottom either in order or in a random order. This method is very organized.

Tables <table>, <tr>, <th>, <td>

Description:

Tables are utlized in websites to organize and format attributes. This method is very beneficial because it allows visitors to access things much more easily.

Syntax:

Tables are used withn the body paragraph. The table elements are fitted into a semi-complex structure, but is separated into organized columns. The table heading(th) and table data(td) are inside the tablet rows(tr). And the table rows are all fitted into the table(tablet).

Example:

<!DOCTYPE>
<html>
<body>
<table>
<tr>
<th>(heading)</th>
</tr>
<tr>
<td>(data)</td>
</tr>
</table>
</body>
</html>

Display:

The table displays many columns with organized lines, which separate rows and topics.

Blocks <div> & <span>

Description:

Block can be used as a styling and formatting technique. You separate words into its own chunk and change the style.

Syntax:

Blocks are used in the body and all you have to do is put text inside of the block elements.

Example:

<!DOCTYPE>
<html>
<body>
<div (style, if you decide to add or not)>
(text elements)
(text)
</div>
</body>
</html>

Display:

Blocks are used to make a block background behind text or to color certain text by using blocks. This tool can be used to change the color of attributes such as text.
